使い方や説明などは以下の記事を参照してください。  
https://qiita.com/Y-Shikase/items/2d773dc4d970228437d5

1.入力が msg.payload から msg.text に変わってます。  
2.キャッシュディレクトリに音声ファイルをキャッシュします。この動作により、msg.filename にファイル名を返します。
